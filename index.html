Обучение английскому онлайн

Как
выучить
английкий
в 3 раза быстрее
и увликательно?

Для запоминания английских слов мы используем метод интервальных
повторений, при котором повторение происходит через
неравные промежутки времени.
Так, при изучении блока из 20 слов
сначала повторяется 7-8 раз,
а затем - через 30 секунд.

Затем повторения проводятся через
все более увеличивающиеся промежутки времени.
Таким образом, достигается 97% запоминания за 3 дня
при минимальном числе повторений.
Этот метод можно назвать методом "Антизубрежка".
Никакие другие методики запоминания не дают такого результата!
